Клиент-мод для Minecraft 1.21.4 (Fabric)

v1.0.0
Установка:
1. Скачай .exe — запусти, он сам всё установит
2. Или скачай .jar и кинь в папку .minecraft/mods
3. Запускай через Fabric Loader 1.21.4

Требования: Java 21+, Windows 10/11
